# HSRP Redundancy with Port Security

## Project Overview

This project demonstrates network redundancy and basic network security using HSRP (Hot Standby Router Protocol), Static Routing, and Port Security in Cisco Packet Tracer.

## Concepts Used

- HSRP (Hot Standby Router Protocol)
- Static Routing
- Port Security

## Network Devices

- 3 Cisco 2901 Routers
- 1 Cisco 2960 Switch
- 2 PCs

## Features

- Gateway redundancy using HSRP
- Automatic failover between Active and Standby routers
- Static routing for WAN connectivity
- Port Security configured on switch access ports
- Successful connectivity verification using ping tests

## IP Addressing

Active Router (G0/0) : 192.168.1.1

Standby Router (G0/0) : 192.168.1.2

HSRP Virtual IP : 192.168.1.254

## Verification

- HSRP Active State Verified
- HSRP Standby State Verified
- Port Security Enabled on Switch Ports
- Successful Ping to HSRP Virtual IP (192.168.1.254)
- Successful Ping to Active Router (192.168.1.1)
- Successful Ping to Standby Router (192.168.1.2)
